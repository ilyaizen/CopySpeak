# Requirements: CopySpeak

**Defined:** 2026-03-04
**Core Value:** Double-copy → instant speech must be flawless

## v1 Requirements

### Navigation

- [x] **NAV-01**: User can navigate between Play, Engine, and Settings tabs via header
- [x] **NAV-02**: Active tab is visually highlighted
- [x] **NAV-03**: Tab state is preserved via URL (no reset on tray open/close)

### Engine Page

- [x] **ENG-01**: User can select TTS backend (CLI, ElevenLabs, OpenAI, HTTP)
- [x] **ENG-02**: User can configure backend-specific credentials and command settings
- [x] **ENG-03**: User can test the engine and see pass/fail with specific error diagnosis (command not found, auth failure, stderr, etc.)
- [x] **ENG-04**: User can select voice from the Engine page
- [x] **ENG-05**: Engine page shows inline install guidance when engine is broken (e.g. `pip install kokoro-tts` with copy button)

### Settings Cleanup

- [x] **SET-01**: TTS engine settings are removed from the Settings page (hard delete, not duplication)

### Onboarding

- [x] **OBD-01**: On first launch, app checks if config file exists
- [ ] **OBD-02**: If config is missing, user is redirected to full-screen onboarding page
- [ ] **OBD-03**: Onboarding is non-blocking—user can skip and use the app immediately

### State

- [x] **STA-01**: Draft config changes survive tab switching without being lost

## v2 Requirements

### Engine Enhancements

- [x] **ENG-06**: Voice list fetched from ElevenLabs API (dropdown instead of raw ID field)
- **ENG-07**: Voice preview — play sample audio for selected voice before committing

### Engine Overhaul

- [x] **ENG-09-HTTP-REMOVE**: HTTP engine removed entirely — TtsEngine type is 'local' | 'openai' | 'elevenlabs', HttpTtsConfig deleted, http.rs deleted, migration auto-switches active_backend from 'http' to 'local' with toast
- [x] **ENG-09-CLI-CONSOLIDATE**: CLI preset list consolidated to piper, kokoro-tts, and qwen3-tts — all other presets (chatterbox, coqui-tts, espeak, edge-tts) hard-deleted; install guidance section removed
- [x] **ENG-09-TWO-STAGE**: Two-stage health check for ElevenLabs and OpenAI — "Check API Key" (credential-only, no synthesis) + "Test Engine" (full synthesis); manual button only, never auto-run
- [x] **ENG-09-OPENAI-VOICE**: OpenAI voice field replaced with static dropdown of all 9 official voices: alloy, ash, coral, echo, fable, nova, onyx, shimmer, verse

### Extended Onboarding

- **OBD-04**: First-run wizard for completely fresh installs (no config at all)
- **OBD-05**: Per-app whitelist/blacklist configuration

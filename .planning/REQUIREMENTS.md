# Requirements: CopySpeak

**Defined:** 2026-03-26
**Core Value:** Double-copy → instant speech must be flawless

## v0.2 Requirements

Requirements for Settings Consolidation milestone. Each maps to roadmap phases.

### Tab Structure

- [ ] **SETT-01**: User sees General tab with Startup, Appearance, Playback, HUD (minified select), History, Triggers, Hotkeys sections
- [ ] **SETT-02**: User sees Advanced tab with Pagination (minified select), Sanitization sections
- [ ] **SETT-03**: User sees About tab with App info, Import/Export sections

### Layout

- [ ] **SETT-05**: Allsettings use 2-column grid layout (Label+InfoTip on left, control on right)
- [ ] **SETT-06**: Pagination settings consolidated into single select dropdown (Disabled, 200, 400, 600, 800, 1000, 1200, 1400, 1600, 1800, 2000)
- [ ] **SETT-07**: HUD settings shown as select dropdown with Disabled option followed by position options

### Navigation

- [ ] **SETT-08**: Settings sidebar shows 3 navigation items instead of 8
- [ ] **SETT-09**: Active section highlight updates correctly when scrolling within consolidated tabs

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature                     | Reason                                             |
| --------------------------- | -------------------------------------------------- |
| Settings data model changes | No backend changes; purely frontend reorganization |
| New settings added          | Only reorganizing existing settings                |
| Settings search/filter      | Not in this milestone                              |
| Mobile-responsive settings  | Desktop-only app                                   |
